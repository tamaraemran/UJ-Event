# UJ Event - University of Jeddah Events Platform

**UJ Event** is a comprehensive Full-Stack web solution developed to centralize university activities. It bridges the gap between event organizers and students through a seamless digital experience.

##  Features
* **Dynamic Event Dashboard:** Real-time display of upcoming events fetched from a MySQL database.
* **Event Registration:** Secure student registration system with duplicate entry prevention.
* **Event Management:** Authorized form to add new events with date and location validation.
* **Contact System:** Integrated communication form with server-side safety checks.
* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.

##  Technical Stack
* **Frontend:** HTML5 (Semantic Structure), CSS3, JavaScript (ES6+).
* **Backend:** Node.js with Express.js framework.
* **Database:** MySQL (Relational Database Management).
* **Validation:** Express-Validator (Custom Middlewares).
* **Icons:** Font-Awesome 6.0.

##  Architecture
The project follows a **Client-Server Architecture**:
1.  **Client-Side:** Handles user interaction and sends asynchronous HTTP requests (AJAX).
2.  **Server-Side:** Processes business logic, validates data patterns, and communicates with the DB.
3.  **Persistence Layer:** MySQL stores events, contact messages, and registrations safely.
